# Blog-Template
A full-stack implementation of an easy to use blog! Uses Express.js for backend, and vanilla JS for frontend.  

Features include:
- GET /posts (retrive posts from the JSON file)
- PUT /posts/<id> (edits an existing post)
- DELETE /posts/<id>
- Login page with JWT auth

I might update later with more features as well like tagging and search functionality for the frontend. 

Adapt as you want!
